The agent app streamlines event workflow. Users use it to manage the process from creating event post, ticketing, to after-event recap.

Access the agent at: https://event-agent-service-139431326628.us-central1.run.app

Local Run:

*  Find API Keys at https://aistudio.google.com/api-keys
*  echo 'GOOGLE_API_KEY="KEY_HERE"' > .env
*  `adk web --port 8000`


Docs:
*  https://google.github.io/adk-docs/get-started/python/
*  https://google.github.io/adk-docs/deploy/cloud-run/
